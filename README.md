# Design Pattern Examples

This repository contains examples of design patterns implemented in Java. Design patterns are proven solutions to common problems in software design.

## Project Structure

- `src/main/java`: Main source code
- `src/main/resources`: Resources
- `src/test/java`: Test code
- `src/test/resources`: Test resources

## Design Patterns to be Implemented

- Creational Patterns
- Structural Patterns
- Behavioral Patterns

## How to Run

```bash
# Build the project
mvn clean install

# Run tests
mvn test
```
